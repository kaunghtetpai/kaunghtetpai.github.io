Accessibility Statement

Last updated: January 9, 2025

Our Commitment: We are committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

This website is partially conformant with WCAG 2.1 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Accessibility Features

We have implemented the following accessibility features:

Keyboard Navigation

  • All interactive elements are keyboard accessible
  • Skip-to-content link for screen reader users
  • Logical tab order throughout the website
  • Visible focus indicators on all interactive elements

Visual Design

  • High contrast color scheme meeting WCAG AA standards
  • Scalable text that can be enlarged up to 200% without loss of functionality
  • Consistent navigation and layout structure
  • Clear visual hierarchy with proper heading structure

Content Structure

  • Semantic HTML markup for screen readers
  • Proper heading hierarchy (H1, H2, H3, etc.)
  • Descriptive link text
  • Alternative text for images
  • Form labels and error messages

Interactive Elements

  • Minimum touch target size of 44x44 pixels
  • Clear focus states for all interactive elements
  • Descriptive button and link text
  • Form validation with clear error messages

Known Issues

We are aware of the following accessibility issues and are working to address them:

  • Some decorative animations may not respect reduced motion preferences
  • Custom cursor effects are not accessible to screen reader users
  • Some third-party embedded content may not be fully accessible

Assistive Technologies

This website has been tested with the following assistive technologies:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Keyboard-only navigation
  • Voice recognition software
  • Screen magnification software

Browser Compatibility

This website is designed to be accessible across modern browsers including:

  • Chrome (latest version)
  • Firefox (latest version)
  • Safari (latest version)
  • Edge (latest version)

Mobile Accessibility

Our website is designed to be accessible on mobile devices with:

  • Responsive design that works on all screen sizes
  • Touch-friendly interface with appropriate target sizes
  • Support for mobile screen readers
  • Optimized performance for slower connections

Third-Party Content

We use some third-party services that may not be fully accessible:

  • Google Fonts for typography
  • Font Awesome for icons
  • GSAP for animations
  • Three.js for 3D graphics

We are working with these providers to ensure better accessibility support.

  • Your preferred method of contact for our response

  • Regular accessibility audits and testing
  • Staff training on accessibility best practices
  • User testing with people with disabilities
  • Staying current with accessibility standards and guidelines
